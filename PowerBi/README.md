# Retail Sales Dashboard (Power BI)

## 📌 Project Overview
This project analyzes retail sales performance to identify trends in revenue, customer behavior, and product categories.

## 🎯 Business Problem
Retail managers need insights into:
- Which products drive the most revenue
- Seasonal sales trends
- Customer purchasing behavior

## 📊 Key Insights
- Sales peak during Q4 due to holiday demand
- Category X contributes 45% of total revenue
- Repeat customers generate 60% of sales

- Power BI
